chip8
=====

Compilation
-----------
- SDL2 is required
- compile with: cc -o chip8 *.c -lSDL

Usage
-----
- CHIP-8's Keyboard consists of 16 keys:

-|-|-|-
1|2|3|C
4|5|6|D
7|8|9|E
A|0|B|F

- In the emulator they are mapped respectively as:

-|-|-|-
1|2|3|4
Q|W|E|R
A|S|D|F
Z|X|C|V

---
To download some CHIP-8 games search for "CHIP-8 Games" or "CHIP-8 ROMs"
